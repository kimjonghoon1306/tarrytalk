# 💬 GreenTalk — 채팅 플랫폼

카카오톡 스타일의 웹 채팅 플랫폼입니다.  
**그린 테마** + **낮/밤 모드** 지원

---

## 📁 파일 구조

```
chat-app/
├── index.html       ← 로그인 페이지 (시작점)
├── chat.html        ← 메인 채팅 (단체채팅 + 1대1 채팅)
├── admin.html       ← 관리자 대시보드
├── profile.html     ← 회원 마이페이지
└── README.md        ← 이 파일
```

---

## 🗺 페이지 설명

| 파일 | 역할 | 주요 기능 |
|------|------|-----------|
| `index.html` | 로그인 | 이메일/비번 입력, 각 페이지 링크 |
| `chat.html` | 채팅 메인 | 단체채팅 탭 / 1대1 탭, 메시지 전송, 자동 답장 시뮬레이션 |
| `admin.html` | 관리자 | 통계 대시보드, 회원 테이블, 채팅방 현황, 활동 로그 |
| `profile.html` | 마이페이지 | 정보 수정, 친구 목록, 알림/화면/개인정보 설정 |

---

## ✨ 기능 목록

### 공통
- 🌙☀️ **낮/밤 테마 토글** (모든 페이지에 적용, localStorage 저장)
- 🎨 **에메랄드 그린** 메인 색상 시스템
- 📱 **반응형** 레이아웃

### 채팅 (chat.html)
- 📋 단체채팅 / 1대1 채팅 **탭 전환**
- ✉️ 메시지 전송 (Enter 키 지원)
- 🤖 **자동 답장 시뮬레이션**
- 😊 **이모지 퀵바** (클릭으로 삽입)
- 📎 파일/이미지/음성 도구바
- 👥 오른쪽 패널: 멤버 목록, 공유 파일
- 🔴 **읽지 않은 메시지 뱃지**

### 관리자 (admin.html)
- 📊 4가지 **핵심 통계 카드** (회원수, 온라인, 메시지, 신고)
- 👤 **회원 테이블** (수정/삭제/상태 관리)
- 📈 **메시지 바 차트** (요일별)
- 🗂 **채팅방 인기 순위**
- 📋 **실시간 활동 로그**

### 마이페이지 (profile.html)
- 🖼 **프로필 히어로** (배경 + 통계)
- ✏️ **정보 수정** (이름, 이메일, 상태, 비밀번호)
- 👥 **친구 목록** + 바로 메시지 전송
- 🔔 **알림 설정** 토글
- 🖥 **화면 설정** (다크모드, 읽음표시 등)
- 🔒 **개인정보 설정**

## 🚀 다음 개발 단계

1. **백엔드 연동** — Node.js + Socket.io (실시간)
2. **DB 연결** — MongoDB or PostgreSQL
3. **인증** — JWT + OAuth (카카오/구글 로그인)
4. **PWA 변환** — manifest.json + Service Worker → 앱처럼 설치 가능
5. **React Native 포팅** — 동일한 디자인으로 iOS/Android 앱 제작
6. **파일 업로드** — Multer + S3 연동

---

## 📲 웹→앱 전환 계획

```
웹 (현재) → PWA (설치형 웹앱) → React Native (네이티브 앱)
```

- PWA: `manifest.json` + `service-worker.js` 추가만으로 홈화면 설치 가능
- React Native: 동일 컴포넌트 구조로 iOS/Android 동시 개발
